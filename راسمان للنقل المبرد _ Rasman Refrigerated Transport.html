تأسست في عام 2020 Established in 2020

تأسست شركة راسمان للنقل المبرد في عام 2020 وسرعان ما أصبحت واحدة من الشركات الرائدة في قطاع النقل المبرد في المملكة. نحن نقدم خدمات نقل موثوقة تلبي احتياجات مختلف القطاعات مع الالتزام بأعلى معايير الجودة والسلامة. Rasman Refrigerated Transport was established in 2020 and quickly became one of the leading companies in the refrigerated transport sector across the Kingdom. We provide reliable transportation services that meet the needs of various sectors while adhering to the highest quality and safety standards.

تقديم خدمات نقل مبرد آمنة وموثوقة تلبي احتياجات عملائنا مع الحفاظ على جودة وسلامة المنتجات المنقولة طوال الرحلة. To provide safe and reliable refrigerated transport services that meet our customers' needs while maintaining the quality and safety of transported products throughout the journey.

GitHub Pages Upload Instructions

Step 1: Create a GitHub Account

If you don't have one, go to GitHub.com and sign up.

Step 2: Create a New Repository

  1. Click the "+" icon in the top right corner and select "New repository"
  2. Name your repository (e.g., "rasman-transport")
  3. Make it Public (required for GitHub Pages)
  4. Click "Create repository"

Step 3: Upload This Website File

  1. In your new repository, click "Add file" → "Upload files"
  2. Drag and drop this HTML file (rename it to index.html if needed)
  3. Click "Commit changes"

Step 4: Enable GitHub Pages

  1. Go to repository "Settings" (tab near the top)
  2. In the left sidebar, click "Pages"
  3. Under "Source", select "Deploy from a branch"
  4. Select "main" branch and "/" (root) folder
  5. Click "Save"

Step 5: Access Your Live Website

After a few minutes, your website will be available at:

https://[your-username].github.io/[repository-name]/

Additional Tips

  • You can use a custom domain by configuring it in the GitHub Pages settings
  • Any changes you make to the repository will automatically update the website
  • For more complex websites, consider using a static site generator or CMS
